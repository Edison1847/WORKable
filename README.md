# 🛰️ WORKable · Human System Intelligence (HSI)

**WORKable** is a high-fidelity intelligence platform designed to convert subjective workforce experience into objective operational alpha. By capturing ground-truth data directly from the organizational source, it provides leadership with real-time visibility into value leaks, manager effectiveness, and human system maturity.

---

## 🏗️ Platform Architecture

The system operates across three core specialized environments that must be running simultaneously to ensure full data-to-insight synthesis:

### 1. 📟 Backend Intelligence Engine (Node.js)
The central nervous system of the platform. It handles the SQLite data layer, diagnostic processing, and NLP sentiment analysis.
- **Port:** 3000
- **Command:** `cd server && node server.js`

### 2. 📊 Strategic Analytics Dashboard
The executive "Cockpit." A high-fidelity, glassmorphism-inspired interface for deep organizational reveals and ROI-driven action plans.
- **Port:** 5173
- **Command:** `cd main-dashboard && npm run dev`

### 3. 📡 Enterprise Diagnostic Intel Node
The field-capture interface. A streamlined, high-performance environment for employee audits and supervisor diagnostics.
- **Port:** 5174
- **Command:** `cd l1-intake && npm run dev`

---

## 🛠️ Performance Features
- **Ground-Truth Archival**: One-click "Strategic Snapshots" for full organizational data versioning.
- **NLP Sentiment Radar**: Real-time detection of organizational "Dark Matter" and crisis proximity.
- **Value Leak Analysis**: Automated translation of workforce friction into quantified salary impact.
- **Maturity Intelligence**: Board-ready metrics for human capital longevity and capability growth.
